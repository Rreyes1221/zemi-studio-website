# Zemi Studio website

Static marketing website for Zemi Studio in Mooresville, North Carolina. The site is designed for free hosting on GitHub Pages while Acuity Scheduling continues to handle appointments and payments.

## Included

- Responsive single-page website
- Service menu and pricing
- Links to Acuity, Instagram, TikTok, phone, and directions
- Local business structured data
- Sitemap, robots file, canonical URL, and social sharing metadata
- GitHub Pages custom-domain file for `zemistudios.com`

## Before publishing

1. Review all business details, prices, and hours.
2. Publish the repository using GitHub Pages from the `main` branch.
3. Set the Pages custom domain to `zemistudios.com`.
4. Update the Squarespace-managed DNS only after the GitHub Pages preview has been approved.
5. Enable HTTPS after DNS validation completes.

The site does not collect or process customer information. Booking and payment links open the existing Acuity Scheduling page.
